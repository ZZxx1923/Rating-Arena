# 🚀 دليل رفع مشروع Arena أونلاين (مجاناً)

لجعل مشروعك يعمل من الجوال والكمبيوتر في أي مكان، سنقوم بتقسيم الرفع إلى جزئين:

---

## 1️⃣ الجزء الأول: رفع السيرفر (Backend) على Render
منصة Render تدعم Node.js مجاناً.

1. اذهب إلى [Render.com](https://render.com) وأنشئ حساباً.
2. اضغط على **New +** ثم اختر **Web Service**.
3. قم برفع مجلد المشروع (أو اربطه بـ GitHub).
4. **الإعدادات الهامة:**
   - **Runtime:** `Node`
   - **Build Command:** `npm install`
   - **Start Command:** `node server.js`
5. **إضافة متغيرات البيئة (Environment Variables):**
   اذهب لتبويب **Environment** وأضف البيانات التالية (بيانات قاعدة بيانات SQL الخاصة بك):
   - `DB_HOST`: رابط قاعدة البيانات (مثلاً `db.example.com`)
   - `DB_USER`: اسم المستخدم
   - `DB_PASSWORD`: كلمة المرور
   - `DB_NAME`: اسم قاعدة البيانات (`employee_system`)
   - `PORT`: `10000` (سيقوم Render بتوفيره تلقائياً)

6. بعد الرفع، سيعطيك Render رابطاً مثل: `https://arena-server.onrender.com`. **احفظ هذا الرابط.**

---

## 2️⃣ الجزء الثاني: تحديث الواجهة الأمامية
قبل رفع الواجهة على Netlify، يجب إخبارها بمكان السيرفر الجديد.

1. افتح ملف `dashboard/index.html`.
2. ابحث عن سطر `const API = ...` (تقريباً في السطر 404).
3. استبدل الرابط برابط السيرفر الذي حصلت عليه من Render:
   ```javascript
   const API = 'https://arena-server.onrender.com';
   ```

---

## 3️⃣ الجزء الثالث: رفع الواجهة (Frontend) على Netlify
منصة Netlify هي الأسهل لرفع الملفات الثابتة.

1. اذهب إلى [Netlify.com](https://netlify.com).
2. سجل دخولك واذهب لتبويب **Sites**.
3. قم بسحب وإسقاط مجلد المشروع بالكامل في منطقة الرفع.
4. مبروك! سيعطيك Netlify رابطاً لموقعك يعمل الآن من الجوال والكمبيوتر.

---

## 💡 ملاحظات هامة:
- **قاعدة البيانات:** تأكد أن قاعدة بيانات SQL الخاصة بك تسمح بالاتصال الخارجي (Remote Access) ليتمكن سيرفر Render من الوصول إليها.
- **كود التفعيل:** لا يزال كود إنشاء الحسابات هو `Arena2026`.
- **الدب التفاعلي:** سيعمل الآن بسلاسة أونلاين.

بالتوفيق في إطلاق مشروعك! 🎉
